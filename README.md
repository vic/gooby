Gooby 
====

An experimental rubinius bytecode interpreter in Go.


![Gooby](https://raw.githubusercontent.com/vic/gooby/master/gooby.jpg "Gooby")


```shell
goop go run main/gooby.go compile test/hello.rbc
```
